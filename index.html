Math Trainer

This page is created to train your math skills (digit sum, multiplication table, small and big squared numbers/roots).

How to use this page

  1. Use the sliders below to change the problem type distribution
  2. Type your answer in the input after the question
  3. Press Enter, Shift or Ctrl to check your answer
    1. If correct the next question appears
    2. If wrong hover the question to reveal the answer
